# 🎬 Movie Recommendation System

A content-based Movie Recommendation System built using Python, Machine Learning, and Streamlit. The application recommends similar movies based on user selection and displays movie posters using the TMDB API.

## 🚀 Features

* Recommend movies based on content similarity
* Interactive web interface using Streamlit
* Movie poster integration using TMDB API
* Fast and user-friendly recommendations
* Real-time recommendation generation

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Pickle
* Requests
* TMDB API

## 📊 Project Workflow

1. Data Collection and Preprocessing
2. Feature Engineering
3. Similarity Matrix Generation
4. Recommendation Engine Development
5. Streamlit User Interface
6. TMDB API Integration for Posters

## 🎯 How It Works

* User selects a movie from the dropdown menu.
* The system identifies similar movies using a precomputed similarity matrix.
* Top recommended movies are displayed along with their posters.
* Movie poster information is fetched dynamically using the TMDB API.
